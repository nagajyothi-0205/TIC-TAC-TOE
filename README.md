# 🎮 Tic Tac Toe Game

A simple and interactive **Two-Player Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. The game runs directly in a web browser without requiring any additional libraries or frameworks.

## 📌 Features

* 👥 Two-player mode (Player X vs Player O)
* 🎨 Modern and responsive user interface
* ✨ Interactive gameplay with hover effects
* 🏆 Automatically detects the winner
* 🤝 Detects draw situations
* 🔄 Restart Game button
* 📱 Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it using any modern web browser (Google Chrome, Microsoft Edge, Firefox, Safari, etc.).
4. Start playing with two players.

## 🎯 How to Play

1. The game starts with **Player X**.
2. Players take turns clicking an empty square.
3. The first player to place **three identical symbols (X or O)** in a row, column, or diagonal wins.
4. If all nine cells are filled without a winner, the game ends in a draw.
5. Click the **Restart Game** button to play again.

## 📖 Game Rules

* Only one move is allowed per cell.
* Players alternate turns.
* The game automatically checks for a winner after every move.
* Once the game ends, no further moves can be made until it is restarted.
